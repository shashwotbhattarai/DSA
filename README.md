## Running a TypeScript File

1. Install dependencies:

   ```sh
   cd typescript
   npm install
   ```

2. Run a TypeScript file using `ts-node`:

   ```sh
   npx ts-node src/slidingWindow/Maximum-Sum-Subarray-of-Size-K.ts
   ```

   Replace the path with your desired `.ts` file as needed.
